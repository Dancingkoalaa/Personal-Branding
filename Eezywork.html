EezyWork

Mijn eerste echte opdracht binnen een bedrijf. Van Augustus 2022 tot Januari 2023 heb ik stage gelopen in een klein ict bedrijf gevestigd in naaldwijk. Dit bedrijf was genaamd Onwego, waar ik samen met andere stagiars heb gewerkt om de Eezywork website uit te breiden voor een officiele opdrachtgever. Eezywork is een erg uitgebreid planningsysteem wat wordt gebruikt binnen het westland. Gebruikers kunnen hier met eigen profielen alle medewerkers inplannen en zorgen dat zij alle benodigde materialen hebben voor de klus. Dit project is zeker een erg speciaal iets voor mij geweest. Niet alleen omdat ik hier 4 maanden aan heb gewerkt, maar ook omdat ik bij dit project heb ontdekt wat mijn sterke en zwakke punten zijn. En ook heb ik skills ontwikkeld als het leiden van een team en het versterken van mijn debugging kennis.

Talen en Showcase

Dit project is gemaakt met HTML, CSS, PHP en Javascript. Verder is er gebruik gemaakt van Axios en Nuxt zodat alle code kon communiceren met de API. Ook is er gebruik gemaakt van Cypress om te zorgen dat alles binnen de website volledig kon worden getest
Helaas. Op verzoek van de klant kan ik de site niet laten zien. Op dit moment ben ik nog bezig met overleggen wat ik eventueel kan laten zien. Ik hoop in de toekomst diverse onderdelen te kunnen laten zien waar ik aan heb gewerkt. Zoals het toevoegen van afbeeldingen/documenten, het maken van modals en kleinere andere toevoegingen
