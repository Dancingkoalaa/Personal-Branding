That Stupid GymLeader!!

That Stupid GymLeader is my first ever react native application. The application is used for pokemon go players. They can add information about a gym to the app to inform other players. This information can be things like: Most used pokemon, well known attackers and also fun facts about the gym. The application also has a map with all gym locations pinned on the map. an other feature that the application has is a light/dark mode depending on the users preference and different languages.

Language and Showcase

This project was made with React and Javascript
