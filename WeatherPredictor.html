Weather Predictor

The weather predictor was a small ML5 project, it uses data found from other sources and learns from the data. With all the learned data, the application will make a neural network. This is used when you fill in the form. The application will then tell you about the weather it thinks that will come that day and provide advice for the way to travel around. Because the application was ment for a bigger project and we had to work a lot with sustainability and minimum emission, the application will avoid the car. It will provide any other possible travel option that the city of Rotterdam has to offer.
