About me

Hi! Ik ben Tim, of voor sommige vrienden Koala. Ik ben een student aan de Hogeschool in Rotterdam. Ik heb altijd al een interesse gehad in technologie wat over de jaren is gegroeid. Waarin mijn focus is gelegd op full-stack development De reden hiervoor is dat ik het ontwerpen erg interessant vind maar toch ook een grote interesse heb in back-end. Mede door mijn voorgaande kennis, samen met de kennis die ik heb gekregen bij CMGT heb ik veel diverse talen en frameworks geleerd. Dit zijn diverse talen zoals React native, typescript, Laravel en Cypress. Maar ook de bekendere talen als HTML, CSS en Javascript staan binnen mijn kennisgebied.

Persoonlijk zou ik mijzelf zien als een erg kritisch en realistisch persoon. Wat ik zelf erg belangrijk vind bij grote projecten. Vooral wanneer ik wordt gevraagd om feedback te geven of om iets te bekijken neem ik hier de tijd voor om het project en/of document volledig te bekijken. Probeer kritische vragen te stellen om hiervoor de beste hulp te bieden die ik kan. Verder heb ik recentelijk mijn ondernemende skills erg ontwikkeld en ben ik erg gegroeid als een projectleider en debugger

voordat ik begon op het Hbo heb ik ee opleiding gedaan als MBO niveau 4 developer, waar ik basiskennis heb opgedaan voor ICT. Hier heb ik vooral gewerkt met HTML, CSS, Javascript en PHP. Waardoor ik de conclusie heb getrokken dat dit zeker mijn gewenste werkgebied is.

Ik hoop na mijn opleiding te kunnen werken als een full-stack ontwikkelaar waar ik focus op front-end en back-end. Mijn hoop is om vooral te werken als webdeveloper met een kleinere focus op mobiel development. Mijn echte einddoel zou zijn om een ProjectManager te zijn en hier nog veel verder in te groeien
